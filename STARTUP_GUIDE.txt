🎤 HOW TO USE VOICE COMMANDS
═══════════════════════════════════════════════════════════

SUPPORTED VOICE COMMANDS:
─────────────────────────

Format: "[Party] [Material] [Vehicle] [Quantity] [Trips] [Rate] [Payment]"

EXAMPLES:

1. "Ramesh reti tractor 6 trip 1200 udhar"
   → Party: Ramesh
   → Material: reti (sand)
   → Vehicle: tractor
   → Quantity: 6 (tons)
   → Trips: 1 (one trip / implied)
   → Rate: ₹1200
   → Payment: udhar (pending/credit)

2. "Amit balu truck 10 800 paid"
   → Party: Amit
   → Material: balu (sand)
   → Vehicle: truck
   → Quantity: 10 tons
   → Rate: ₹800 per ton
   → Payment: paid (cash/immediate)

3. "Priya stone 5 1500 advance"
   → Party: Priya
   → Material: stone
   → Quantity: 5 tons
   → Rate: ₹1500 per ton
   → Payment: advance (partial payment)

SUPPORTED MATERIALS:
───────────────────
• reti / sand / balu
• mitti / soil
• stone / gravel
• cement
• brick
• Custom: Add via "Manage Materials"

SUPPORTED VEHICLES:
──────────────────
• tractor
• truck
• tempo
• chhota (small truck)
• bada (big truck)
• Custom: Add via "Manage Vehicles"

PAYMENT STATUS:
──────────────
• udhar = Pending (Credit)
• paid = Paid (Cash)
• advance = Partial Payment

STEPS TO USE:
─────────────
1. Go to Dashboard (http://localhost:5000/dashboard/)
2. Click "🎤 Start Recording" button
3. Speak your transaction:
   "Ramesh reti tractor 6 trip 1200 udhar"
4. Parsed data appears in text field
5. Review the extracted information
6. Click "Create Transaction from Voice"
7. Transaction saved! Invoice generated automatically

⚠️  ERROR TROUBLESHOOTING
═══════════════════════════════════════════════════════════

"Flask module not found"
→ Activate virtual environment: ..\.venv\Scripts\Activate.ps1
→ Reinstall: pip install -r requirements.txt

"Port 5000 already in use"
→ Option 1: Edit run.py, change port=5000 to port=8000
→ Option 2: Kill existing process:
   netstat -ano | findstr :5000
   taskkill /PID <PID> /F

"Speech recognition not working"
→ Use Chrome, Edge, or Safari
→ Enable microphone permissions
→ Check browser console (F12) for errors
→ Ensure voice language is set correctly

"Can't access from phone"
→ Check phone & laptop on same Wi-Fi
→ Verify laptop IP: ipconfig
→ Check Windows Firewall allows port 5000
→ Try: http://<IP>:5000 (not https://)

"Database locked"
→ Stop application
→ Delete: accounting_system.db
→ Restart application (will recreate)

"Session timeout issues"
→ Edit app/__init__.py
→ Change: PERMANENT_SESSION_LIFETIME = 1800  (in seconds)

"PDF generation failing"
→ Ensure invoices/ folder exists
→ Check folder permissions
→ Verify reportlab is installed: pip install reportlab
